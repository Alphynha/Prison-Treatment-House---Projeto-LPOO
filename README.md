# Sistema Penitenciário (Java Console)

Projeto desenvolvido em **Java** com foco na aplicação de **Programação Orientada a Objetos (POO)**, utilizando uma aplicação **console** simples, sem interface gráfica e sem banco de dados.

O objetivo do projeto é modelar um sistema penitenciário básico, permitindo o gerenciamento de **presidiários**, **celas** e **visitas**.

---

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Herança
- Classes abstratas
- Enums
- Separação de responsabilidades
- Camada de serviço (Service)
- Repositório em memória (ArrayList)
- Tratamento de exceções
- Aplicação em console

---

## Funcionalidades

### Presidiários
- Cadastrar presidiário
- Listar presidiários
- Alocar presidiário em cela
- Desalocar presidiário da cela

### Celas
- Cadastrar cela
- Listar celas
- Controle automático de ocupação
- Status da cela:
  - DESOCUPADA
  - OCUPADA
  - LOTADA

### Visitas
- Registrar visita para um presidiário
- Listar visitas por presidiário
- Listar todas as visitas

## Como executar

1. Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git

3. Execute a classe Main

4. Utilize o menu no console para interagir com o sistema
